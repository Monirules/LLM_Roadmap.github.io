# NLP to LLM Roadmap

This repository presents a structured **7-week learning roadmap** for moving from classical Machine Learning toward modern **Natural Language Processing (NLP), Large Language Models (LLMs), Multimodal Large Language Models (MLLMs), and Vision-Language Models (VLMs)**.

It is designed for learners who already have basic Machine Learning knowledge and want to prepare for **PhD-level research in NLP, LLMs, and multimodal AI**.


## Live Preview

Live Preview - https://monirules.github.io/LLM_Roadmap.github.io/ 


## What This Roadmap Covers

- A step-by-step weekly learning plan from **PyTorch basics** to advanced LLM and VLM research topics.
- Theory, coding practice, research papers, mini-projects, and weekly milestones.
- Core topics such as **Transformers, Hugging Face, BERT, GPT, LoRA, QLoRA, RAG, CLIP, LLaVA, and multimodal models**.
- **Progress tracking**, must-read papers, reading resources, project ideas, compute guidance, and daily study planning.

## Roadmap Focus

This roadmap is organized to help learners gradually build the required foundation for modern NLP and LLM research:

1. Deep Learning and PyTorch Foundations  
2. NLP Fundamentals and Text Representations  
3. Transformer Architecture  
4. Pretrained Models and Hugging Face Ecosystem  
5. Large Language Models  
6. Multimodal LLMs and Vision-Language Models  
7. Mini-Project and Research Orientation
